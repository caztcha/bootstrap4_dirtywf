ナビゲーションメニュー (スクロール)

画面幅に収まりきらない場合は横方向にスクロールする、ナビゲーションメニューです。

<nav> 要素のすぐ内側で <div class="container"> を有効にすると、コンテンツを container 幅に収めることができます。

アクセシビリティの配慮

  • キーボード操作 ([Tab] キーによるフォーカスおよび [Enter] キーによる実行) が可能です。
  • <nav> 要素に aria-label (="メインメニュー") を記述し、ナビゲーションの種類を明示しています。

この UI コンポーネントに固有の CSS および JavaScript

特記事項なし。
