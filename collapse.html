折り畳み & 展開

詳細情報を折りたたんでいる (必要に応じて開閉できる) UI です。

アクセシビリティの配慮

  • キーボード操作 ([Tab] キーによるフォーカスおよび [Enter] キーによる実行) が可能です。
  • 開閉状況に応じて aria-expanded 属性値が変わります (true/false)。
  • ボタンのサイズを 44 × 44 CSS px 確保しています。
  • Nice-to-have 機能として、見出しのクリック/タップでも、開閉が可能です。

この UI コンポーネントに固有の CSS および JavaScript

  • <head> 内の Additional CSS (for Collapse/Expand)
  • </body> 手前の Additional JavaScript (for Collapse/Expand)
