折り畳み & 展開

詳細情報を折りたたんでいる (必要に応じて開閉できる) UI です。

アクセシビリティの配慮

  • [Tab] キーによるフォーカス移動および [Enter] キーによる実行 (折り畳み/展開の切替) が可能。
  • WAI-ARIA の aria-expanded 属性を入れることで、スクリーンリーダーユーザーにも開閉状況を伝えている。

固有の CSS および JavaScript

  • <head> 内の Additional CSS (for Collapse) にある collapse.css。
