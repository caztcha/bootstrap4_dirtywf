フッター (含ビッグフッター)

サイト共通フッターです。

ビッグフッター部分はレスポンシブデザインに対応していて、画面幅が Bootstrap の "Medium" 以下 (992px未満) では、メニューが折り畳まれます。

アクセシビリティの配慮

  • 画面幅が "Medium" 以下の際、メニューの開閉がキーボード操作でも可能です。また、開閉に応じて aria-expanded の値が変わります (false/true)。
  • ビッグフッター部分 (<nav> 要素) に「aria-label="サイトマップ"」を記述し、スクリーンリーダー利用者に対してナビゲーションの種類を明示しています。

固有の CSS および JavaScript

  • HTML の中にインラインでスタイルを調整しています。
  • </body> 手前の Additional JavaScript (for Big Footer)。画面幅に応じた class 適用を調整しています。
