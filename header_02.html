ヘッダー (#02)

レスポンシブなメガドロップダウンメニューです。

<nav> 要素のすぐ内側で <div class="container"> を有効にすると、ヘッダー内のコンテンツを container 幅に収めることができます。

アクセシビリティの配慮

  • ドロップダウンメニューをキーボード操作で開閉できます。Bootstrap のドロップダウンメニューの標準仕様として、展開中のメニューからフォーカスが外れると、自動的にそのメニューを閉じます。(複数のドロップダウンメニューが共存する場合の競合防止。)
  • ドロップダウンメニューの開閉に応じて aria-expanded の値が変わります (false/true)。
  • <nav> 要素に「aria-label="サイト全体ナビゲーション"」を記述し、スクリーンリーダー利用者に対してナビゲーションの種類を明示しています。
  • サイト内検索の <form> 要素に「role="search"」を記述しています。

この UI コンポーネントに固有の CSS および JavaScript

  • <head> 内の Additional CSS (for Mega Dropdown Menu)。
  • </body> 手前の Additional JavaScript (for Mega Dropdown Menu)。展開中のメニューエリア内をクリックしたときにドロップダウンが閉じてしまう現象を解消しています。
