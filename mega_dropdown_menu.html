メガドロップダウンメニュー

レスポンシブなメガドロップダウンメニューです。

アクセシビリティの配慮

  • キーボード操作の担保。Bootstrap のドロップダウンメニューの標準仕様として、展開中のメニューからフォーカスが外れると、自動的にそのメニューを閉じる。(複数のドロップダウンメニューが共存する場合の競合防止。)
  • 開閉時のスクリーンリーダー読み上げフィードバック。(aria-expanded)

固有の CSS および JavaScript

  • <head> 内の Additional CSS (for Mega Dropdown Menu) にある mega_dropdown_menu.css。
  • </body> 手前の Additional JavaScript (for Mega Dropdown Menu) にある mega_dropdown_menu.js。展開中のメニューエリア内をクリックしたときにドロップダウンが閉じてしまう現象を解消しています。
