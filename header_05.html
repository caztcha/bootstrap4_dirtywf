ヘッダー (#05)

メニュー + サイトID + サイト内検索、で構成されるヘッダーです。

<nav> 要素のすぐ内側で <div class="container"> を有効にすると、ヘッダー内のコンテンツを container 幅に収めることができます。

アクセシビリティの配慮

  • キーボード操作 ([Tab] キーによるフォーカスおよび [Enter] キーによる実行) が可能です。
  • メニューおよびサイト内検索の開閉状況に応じて aria-expanded 属性値が変わります (true/false)。
  • ボタンのサイズを 44 × 44 CSS px 確保しています。
  • <nav> 要素に aria-label (="サイト全体ナビゲーション") を記述し、ナビゲーションの種類を明示しています。
  • サイト内検索の <form> 要素に「role="search"」を記述しています。

この UI コンポーネントに固有の CSS および JavaScript

特記事項なし。
