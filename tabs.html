タブ

複数パネルを切り替えるタブです。

アクセシビリティの配慮

  • [Tab] キーによるフォーカス移動および [Enter] キーによる実行 (タブ切替) が可能。
  • WAI-ARIA の role (tablist, tab, tabpanel) を入れることで、スクリーンリーダーユーザーにもタブインターフェースであることを伝えている。
  • 小さな画面のスマートフォン (320 CSS pixels) でもタブが横一列に並ぶように、タブの数は2つまたは3つまでとする。タブの数に応じて、タブ幅を style 属性で指定。

固有の CSS および JavaScript

  • <head> 内の Additional CSS (for Tabs) にある tabs.css。
