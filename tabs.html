タブ

複数パネルを切り替えるタブです。

スマートフォンでの利用 (タブのタッチ可能領域の広さの確保、タブ内のテキストの可読性) も考慮し、タブ数は2～3に留めるのがよいでしょう。

アクセシビリティの配慮

  • キーボード操作 ([Tab] キーによるフォーカスおよび [Enter] キーによる実行) が可能です。
  • WAI-ARIA の role (tablist, tab, tabpanel) および aria-controls (tab と tappanel の紐付け) によって、スクリーンリーダー利用者にもタブインターフェースの情報を伝えています。また各 tab に aria-selected属性 (true/false) があることで、どのタブが選択されている状態かも伝えています。
  • 小さな画面のスマートフォン (320 CSS pixels) でもタブが横一列に並ぶように、タブの数は最大で3つとしています。タブの数に応じて、タブ幅を style 属性で指定しています。
